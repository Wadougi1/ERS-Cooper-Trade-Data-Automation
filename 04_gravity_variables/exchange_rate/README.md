# Exchange Rate Data Automation

This submodule retrieves annual exchange rate data to account for currency fluctuations in U.S. agricultural trade.

## Data Sources
- **IMF International Financial Statistics (IFS):** Official exchange rate series.
- **World Bank WDI:** Annual average exchange rates (LCU per US$).

## Usage
These scripts provide the macroeconomic stability indicators used to analyze the competitiveness of U.S. exports in foreign markets.
