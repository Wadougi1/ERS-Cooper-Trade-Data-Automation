# GDP Data Automation

This submodule automates the collection of Gross Domestic Product (GDP) data from multiple international organizations.

## Data Sources
- **IMF DataMapper:** Provides current and projected GDP figures.
- **World Bank API:** Supplies historical GDP (Current US$) series.
- **UN National Accounts:** Offers official government-reported GDP data.

## Usage
Run these scripts to generate a harmonized GDP dataset. The output is used as the "Economic Mass" variable in gravity model estimations.
