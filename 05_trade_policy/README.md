# Trade Policy and Tariff Automation

This module automates the collection of trade policy variables, specifically focusing on **Tariffs** and **Regional Trade Agreements (RTAs)**.

## Research Context

Trade policy variables are critical for analyzing the "resistance" or "facilitation" of international trade flows. 
- **Tariffs:** Represent the direct cost barriers imposed on U.S. agricultural exports.
- **Trade Agreements:** Represent policy frameworks (like FTAs) that reduce or eliminate those barriers, facilitating trade creation.

This module provides the data necessary to conduct scenario analysis on how changes in tariff rates or the signing of new trade agreements impact export performance.

## Module Structure

| Subfolder | Variable | Sources |
| :--- | :--- | :--- |
| `tariffs/` | Applied & MFN Tariffs | World Bank, WITS |
| `agreements/` | Regional Trade Agreements | WTO RTA Database |
